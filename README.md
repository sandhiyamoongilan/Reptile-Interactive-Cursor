# Reptile Interactive Cursor Animation | Web Animation Project

A creative and interactive **reptile-themed cursor animation** built using HTML, CSS, and JavaScript, designed to follow real-time mouse movement and provide an engaging user experience.

---

## Project Overview

This project demonstrates how to build a **custom animated cursor effect** inspired by reptile movement. The animation smoothly tracks the cursor, creating a dynamic and visually appealing UI interaction. It helps in understanding **JavaScript event handling, animation logic, and front-end interaction design**.

---

## Key Features

- Real-time mouse tracking  
- Smooth reptile-style movement animation  
- Clean and responsive UI design  
- Optimized DOM manipulation  
- Lightweight and fast execution  

---

## Tools & Technologies

- HTML5  
- CSS3  
- JavaScript (ES6)  

---

## How to Run

1. Download or clone the repository  
2. Open the `index.html` file in any modern web browser  
3. Move the mouse to experience the interactive reptile cursor animation  

---

## Project Structure

- `index.html` – Main webpage  
- `style.css` – Styling and layout  
- `script.js` – Cursor animation and logic
